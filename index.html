KeyNest (Android Project)

KeyNest is an Android-based password management and generation application designed to securely store and manage user credentials. The app allows users to save passwords for different platforms in an encrypted format and generate strong, secure passwords to enhance online security and also helping people to improve their knowledge on cybersecurity via and inbuilt quiz

Iot-Based Water Level Monitoring (IoT Based)

The IoT-Based Water Level Monitoring System is a smart solution that measures and displays the water level in a tank in real time. It uses an ultrasonic sensor to detect the distance of water from the top of the tank and calculates the water level accordingly. The measured data is shown on an LCD screen, while LED lights indicate different water level conditions (low, medium, and high).

Bubble Pop 2D Casual (Unity Game)

Bubble Pop is a 2D arcade-style game developed using the Unity game engine, where players tap or click on floating bubbles to pop them and score points. The game features smooth animations, increasing difficulty levels, and engaging sound effects to create an addictive and fun gameplay experience.

MonoKeys 2D Platformer (Unity Game)

MonoKeys is a simple 2D platformer where the player must collect three keys to unlock and move to the next level. The game includes basic movement mechanics and scene management for smooth level transitions , reset button for a restarting the scene and pause menu. It was made for a Game Jam Within 3 days
